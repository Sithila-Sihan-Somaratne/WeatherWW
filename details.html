Weather WW Features and Usage



Features



WeatherWW has a lot of features.
Here are some of them:

  • WeatherWW shows the current weather conditions
  • WeatherWW provides a forecast for upcoming days description
    including predicted temperatures, weather conditions, and chance of precipitation.
    The number of days of weather forecast are 10.
  • WeatherWW shows also the historical weather
    (you can only input the date with the past year until the day before)
  • WeatherWW can detect many places with the same city, that will show you an
    alert to see whether one of those results is the one that the user tried to search


Usage



If you want to search
for a city's weather details,
in WeatherWW you can enter the following:

  • Latitude and Longitude (Decimal degree) e.g: 48.8567,2.3508
  • city name e.g.: Paris
  • US zip e.g.: 10001
  • UK postcode e.g: SW1
  • Canada postal code e.g: G2J
  • metar: e.g: metar:EGLL
  • iata:<3 digit airport code> e.g: iata:DXB
  • auto:ip IP lookup e.g: auto:ip
  • IP address (IPv4 and IPv6 supported) e.g: 100.0.0.1
